# Android App Project 
 
This repository contains my Android automation testing project using Appium and Java. 
 
## Overview 
This project demonstrates mobile app test automation using Appium. It includes example scripts and configurations for testing Android apps. 
 
## Tools & Technologies 
- Java 
- Appium 
- Selenium 
- TestNG 
- Eclipse IDE 
 
## How to Run 
1. Clone this repository 
2. Open it in Eclipse 
3. Start Appium server 
4. Run tests via TestNG
